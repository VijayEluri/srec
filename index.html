srec by vtatai

Java Swing record/replay tool, with support for both Java and Ruby / Cucumber tests

srec is a tool for testing Java Swing applications. It supports not only record / replay using a friendly GUI, but also provides a Java DSL for writing Swing tests, as well as a Ruby module to aid in writing Cucumber tests. As a bonus, test scripts can also be run on a Hadoop cluster, speeding up large test suites run.
